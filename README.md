# Crypto-Express

Crypto-Express is a learning-oriented full-stack web application built to practice and improve my skills in modern web development.  
This project is not commercial and is developed as part of my preparation for an internship (Praktikum) and future apprenticeship as a **Fachinformatiker für Anwendungsentwicklung** in Germany.

---

## Purpose of the Project

The main goal of this project is to:
- Strengthen my knowledge of full-stack web development
- Apply backend and frontend concepts in a real project
- Learn how to structure and maintain a real-world application
- Build a portfolio to support my application for IT internships and Ausbildung

I am actively learning during the development process and continuously improving the codebase.

---

##  Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js, Ejs, Bcypt, Jsonwebtoken, Ws, Morgan.
- Database: SQLite3
- Version Control: Git & GitHub

---
